### Flutter cli
- Most used commands:
    - create 'name'
    - doctor
    - run (-d 'deviceid')

---
### Flutter IDE
- Android Studio, version 3.0 or later.
    - Flutter Plugin (run debug, hot reload)
    - Dart Plugin (code validation)
- VS Code, version 1.20.1 or later.
    - Dart code extension
    - Flutter extension

Validate: Flutter doctor from terminal
